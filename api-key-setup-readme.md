# API setup for Torn Bookie Live Scores Panel

Use these steps to create the account, get the exact credential, and paste it into the settings section of the `Torn Bookie Live Scores Panel` script. 

## The Odds API (for betting help - shown in the detail pane)

1. Create your account here: [https://the-odds-api.com/account/](https://the-odds-api.com/account/). 
2. After you register, The Odds API will email your API key by email. 
3. Their terms also say the key may be available in the account dashboard, and if needed you can regenerate a new key in the dashboard at [https://dash.the-odds-api.com/](https://dash.the-odds-api.com/)
4. Copy that API key and paste it into the The Odds API field in the settings section of the `Torn Bookie Live Scores Panel` script. 

## PandaScore (for getting scores on e-sports)

1. Create your account and access the docs here: [https://developers.pandascore.co/docs/getting-started](https://developers.pandascore.co/docs/getting-started). 
2. Open your PandaScore dashboard here: [https://app.pandascore.co/dashboard](https://app.pandascore.co/dashboard).
3. Copy the token from the dashboard and paste it into the PandaScore field in the settings section of the `Torn Bookie Live Scores Panel` script.
